Front-end

The "front end" refers to the user interface of a website or application, encompassing everything a user directly sees and interacts with.

Back-end

The backend, also known as the server-side, is the part of a website or application that users don't directly see or interact with.

Database

A database is an organized collection of data, typically stored electronically, that can be easily accessed, managed, and updated.
